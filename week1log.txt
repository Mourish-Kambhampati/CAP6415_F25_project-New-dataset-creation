Week 1 Log – CAP6415 Fall 2025 Project  
--------------------------------------

This week, I finalized my project idea and set up the structure of the repository.  
The chosen project topic is **Road/Highway Vehicle Detection using a Custom Dataset and YOLOv8**.

Here is what I accomplished during Week 1:

1. **Project Idea Finalization**
   - Decided to create a custom dataset of road/highway images containing various vehicle types such as cars, trucks, buses, motorcycles, and vans.
   - Chose object detection as the main task since it aligns well with the project requirements and allows for multiple objects per image.
   - Selected YOLOv8 as the SOTA model for training and inference because of its high performance, simple workflow, and strong documentation.

2. **Repository Creation**
   - Created the required public GitHub repository named:
     `CAP6415_F25_project-VehicleDetection`.
   - Prepared the repository for future additions such as dataset files, scripts, results, and documentation.
   - Verified that the repository is publicly accessible and ready for updates.

3. **Initial Dataset Planning**
   - Planned out data collection strategy: capture 1000+ images from roadside/highway locations.
   - Identified target classes: car, truck, bus, motorcycle, bicycle, and optionally pedestrian.
   - Decided to collect images during different lighting and environmental conditions to improve dataset robustness.

4. **Preparation for Next Week**
   - Began organizing folders for dataset: raw images, annotations, and processed training data.
   - Installed annotation tool (LabelImg) and tested workflow for bounding box labeling.
   - Prepared to start large-scale data collection and annotation in Week 2.

Overall, Week 1 was focused on planning, setting up the GitHub repository, and preparing the foundation of the project.
